# Hello App

A simple _Hello World Application_ used for studing how to integrate many development tools.

- Source Code Management
    - GitHub
    - Bitbucket
- Continuous Integration
    - CircleCI
    - Codeship
    - DroneIO
    - SnapCI
    - TravisCI
- Test Coverage
    - Coveralls
